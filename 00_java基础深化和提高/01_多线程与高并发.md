# 基本概念
线程：进程中执行的单元，程序中不同的执行路径
# 创建线程/启动线程的方法
```java
public class CreatThread {

    static class MyThead extends Thread{
        @Override
        public void run() {
            System.out.println("Thread Thread Thread");
        }
    }

    static class MyRun implements Runnable{
        @Override
        public void run() {
            System.out.println("Run Run Run");
        }
    }

    public static void main(String[] args) {
        new MyThead().start();
        new Thread(new MyRun()).start();
        new Thread(()->{
            System.out.println("Lambda Lambda Lambda");
        }).start();
    }
    
}
```
